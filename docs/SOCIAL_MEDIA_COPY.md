## 🎓 版本4：专业版（适合医学/科研群）

```
【分享】MedEvidence AI - 循证医学文献智能检索工具

参加魔搭创客松做的小工具，基于 KnowS 医学循证数据库 + StepFun 大模型：

• 支持中英文双语自然语言检索
• 自动识别研究类型并标注 5 级循证等级
  - Level 1: Meta分析 / 高质量RCT（金标准）
  - Level 2: 低质量RCT
  - Level 3: 队列研究 / 病例对照
  - Level 4: 病例系列 / 指南
  - Level 5: 个案报道 / 专家意见
• StepFun LLM 生成中文临床要点摘要

🔗 在线体验：https://www.modelscope.cn/studios/gsym236998/MedEvidence-AI
📦 开源地址：https://github.com/yigenfeng0707-netizen/MedEvidence-AI

欢迎同行试用交流！有好用的需求场景也欢迎提出来～
```
