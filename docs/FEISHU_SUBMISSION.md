### 项目简介（100字以内版本）

支持中英文双语检索、五级循证证据等级自动标注、AI临床摘要生成，帮助医生和科研人员

### 项目亮点（200字版本）
```
MedEvidence AI - 医学循证智能检索助手

核心亮点：
✅ 中英文双语智能检索，一键切换，无需多库切换
✅ 自动识别研究类型，标注 Level 1-5 循证等级
✅ 快速定位 Meta分析、RCT等高等级金标准证据
✅ StepFun 大模型生成中文临床要点摘要
✅ 完全开源，支持魔搭一键部署/Docker/本地运行

技术栈：FastAPI + Python + KnowS API + StepFun LLM
应用场景：临床决策支持、医学研究、文献综述、循证医学教学
```

### 技术方案（详细版 - 500字）
```
【技术架构】
MedEvidence AI 采用前后端分离架构，核心服务基于 FastAPI 构建，
深度集成 KnowS 医学循证数据库 API 和 StepFun 大语言模型 API。

【核心功能实现】
1. 智能检索层
   - 中英文自动检测，智能路由到对应数据库端点
   - 基于关键词提取的相关性评分算法
   - 多源结果融合与去重

2. 循证分级引擎
   - 标题关键词匹配识别研究类型
   - 支持 Meta分析、系统评价、RCT、队列研究等10+种类型
   - 结合期刊声誉和研究设计综合判定证据等级

3. AI摘要生成
   - StepFun step-2-16k 大模型
   - 多文献综述总结 + 临床要点提取
   - 流式输出，渐进式展示

【部署方式】
- 魔搭创空间 Docker 一键部署
- 本地 pip 安装运行
- Docker 容器化部署

【开源协议】
Apache License 2.0
```
