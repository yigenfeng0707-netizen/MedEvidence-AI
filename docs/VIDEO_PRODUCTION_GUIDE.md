## 📹 第一步：录制录屏

### 方案A：使用演示页面录制（推荐，效果最好）

**步骤：**
1. 打开 `docs/demo-video-page.html`（Chrome/Edge 浏览器）
2. 按 `F11` 进入全屏模式
3. 打开 OBS Studio，设置录制参数：
   ```
   分辨率：1920×1080（或 2560×1440）
   帧率：30fps
   格式：MP4 (H.264)
   画质：高（CRF 18-20）
   ```
4. 点击"录制"开始
5. 按下面的时间线操作页面（使用方向键切换场景）

## 🎙️ 第二步：配音制作

### 方案A：AI配音（最快，效果不错）

**推荐工具：**
- 剪映专业版 → 文本朗读 → 选择"解说男声/女声"
- 讯飞配音（xunfei.cn）→ 更自然的AI音色
- Edge TTS（免费）→ 微软语音合成

**操作步骤（剪映）：**
1. 将逐字稿复制到剪映文本
2. 选择"文本朗读"功能
3. 选择合适音色（推荐"沉稳男声"或"知性女声"）
4. 导出音频文件（WAV 或 MP3）

**推荐配音参数：**
- 音色：沉稳专业男声 / 知性女声
- 语速：0.9x（稍慢，更专业）
- 音量：-6dB

### 方案B：真人配音（质量最高）

**准备：**
- 安静的房间
- 领夹麦克风
- 参考逐字稿 `docs/VIDEO_SCRIPT.md`

**录制技巧：**
1. 先通读几遍逐字稿
2. 每段旁白单独录制，方便剪辑
3. 保持语速均匀，每秒钟约 3.5 个字
4. 重音放在关键词上（如"Level 1"、"Meta分析"）
5. 每段录完停顿2秒，方便剪辑

**录制格式：**
```
格式：WAV 或 FLAC（无损）
采样率：48000 Hz
位深：24 bit
声道：单声道（语音）
```

## ✂️ 第三步：剪辑合成

### 使用剪映专业版（推荐，免费且功能强）

#### 3.1 导入素材
1. 打开剪映专业版
2. 新建项目 → 分辨率 1920×1080，帧率 30
3. 导入录屏视频、配音音频、SRT字幕

#### 3.2 时间线排布
```
视频轨1：录屏画面
音频轨1：配音旁白
音频轨2：背景音乐（音量-24dB）
字幕轨1：中文字幕
```

#### 3.3 剪辑要点

**画面与配音对齐：**
- 以旁白为主轴，调整画面切换时机
- 每段旁白说完后，留 0.5-1 秒画面停留
- 场景切换处可加"淡入淡出"转场（0.3-0.5秒）

**录屏节奏调整：**
- 如果录屏太快，可适当放慢（0.8-0.9x）
- 如果录屏太慢，可裁剪冗余部分
- 搜索加载动画可裁剪到和旁白同步

**关键时间节点（参考）：**
```
0:00-0:12  片头Logo
0:12-0:35  痛点引入（3个痛点，每个6-7秒）
0:35-0:50  产品亮相（3个功能卡片依次出现）
0:50-1:30  中文检索演示（输入→加载→5条结果→统计）
1:30-2:05  英文检索演示
2:05-2:30  证据金字塔（5层从下往上构建）
2:30-2:55  技术架构图
2:55-3:10  合规声明
3:10-3:20  结尾页
```

#### 3.4 转场效果
- 场景切换：淡入淡出（0.5秒）
- 卡片出现：上移淡入（0.6秒）
- 数据出现：缩放弹入（0.4秒）
- 整体保持简洁，不要花里胡哨

## 📝 第四步：字幕添加

### 方案A：导入SRT文件（最快）

1. 将 `docs/subtitles_zh.srt` 直接导入剪映
2. 调整字幕样式：
   ```
   字体：思源黑体 Bold / 微软雅黑 Bold
   字号：40-44px（1080p）
   颜色：白色
   描边：2px 黑色（或半透明黑底）
   位置：底部1/6处，居中
   每行字数：≤20字
   ```

### 方案B：自动识别字幕（需校对）

1. 剪映 → 字幕 → 自动识别
2. 选择中文，点击识别
3. **重要：必须校对！** 专业术语可能识别错误
   - "Meta分析" → 检查是否被识别成"mata分析"
   - "RCT" → 检查大小写
   - "循证" → 检查是否是"寻证"

### 字幕同步检查
- 逐句检查字幕与配音是否对齐
- 字幕出现时间比声音早 0.1 秒，消失晚 0.2 秒
- 长句自动拆成两行，每行不超过20字

## 🎵 第五步：音乐音效

### 背景音乐选择
- **风格**：科技感电子乐 / 轻快医疗风
- **来源**：剪映音乐库（搜索"科技"、"医疗"、"科普"）
- **音量**：旁白时 -24dB，无旁白时 -18dB
- **时长**：整曲贯穿，开头结尾做淡入淡出

### 音效点缀（可选，提升质感）
| 场景 | 音效 |
|------|------|
| Logo出现 | "咚" 低音音效 |
| 功能卡片弹出 | "叮" 轻脆音效 |
| 搜索按钮点击 | 点击声 |
| 结果加载 | 轻微数据流音效 |
| 金字塔层级出现 | 递进式音效 |

**注意：** 音效音量要小（-30dB左右），不能盖过旁白。

**分辨率：1920×1080（Full HD）**
```
格式：MP4

帧率：30 fps
码率：VBR，目标 8-12 Mbps

```

**分辨率：3840×2160（4K 超高清）** ← 推荐用于比赛
```
格式：MP4
编码：H.264 (AVC) 或 H.265 (HEVC)
帧率：30 fps
码率：VBR，目标 20-30 Mbps
质量：最高
```

**音频设置：**
```
格式：AAC
采样率：48000 Hz
码率：192 kbps（立体声）
```

### 导出后检查
1. 用播放器全屏观看一遍
2. 检查：
   - [ ] 字幕是否同步
   - [ ] 配音是否清晰
   - [ ] 音乐是否过大/过小
   - [ ] 画面是否清晰
   - [ ] 有没有错别字
   - [ ] 总时长是否符合预期（3分20秒左右）
   - [ ] 结尾链接是否正确

## 💡 加分小技巧

1. **开头3秒抓眼球**：Logo动画要精致，配色要专业
2. **数据可视化**：证据金字塔动效一定要流畅
3. **真实感**：如果可能，穿插1-2秒真实API调用的画面
4. **节奏感**：旁白、画面、音乐三者节奏统一
5. **结尾清晰**：二维码和链接停留足够时间（至少5秒）
6. **缩略图**：视频封面/缩略图设计要吸引点击

## 🆘 常见问题

**Q: 录屏时字体发虚怎么办？**
A: 确保显示器是100%缩放，浏览器放大到125%会让文字更清晰。

**Q: 配音语速怎么控制？**
A: 先录一版，听一下和画面是否匹配。一般中文旁白每秒3-3.5个字最合适。

**Q: 字幕识别不准怎么办？**
A: 用SRT文件导入是最准的。如果用自动识别，专业术语一定要手动改。

**Q: 文件太大传不上去怎么办？**
A: 降低码率（H.265更省空间），或者缩短时长。
